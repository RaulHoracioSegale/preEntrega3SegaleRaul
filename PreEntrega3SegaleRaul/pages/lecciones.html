Colores

Hasta la llegada de CSS3, los colores se podían representar mediante el nombre del color en idioma Ingles, un valor exadecimal, o utilizando los valores RGB que hace referencia a la mezcla de colores "RED" (rojo), "GREEN" (verde) y "BLUE" (azul).
La propiedad "OPACITY" se utilizaba para dar transparencia, pero el problema es que afecta a todo el contenido del contenedor. Esto se resolvió agregando un último valor al final a los valores RGB, denominado canal ALPHA , quedando rgba(), y cuyo valor va de 0 a 1. (Ej: 0, 0.1, 0.2, 0.3, etc.).

En consecuencia, los colores se pueden determinar de las siguientes maneras:

color: #ff0000;

color: rgb(255,0,0);

color: rgba(255,0,0,1);

Ejemplos de transparencia de acuerdo a los valores del canal ALPHA. Tengamos en cuenta que podemos utilizar indistintamente las propiedades "background-color", como "background". Asimismo, en el siguiente ejemplo, vamos del valor 0 (que representa la máxima transparencia) hasta el valor 1 (que representa al color puro).

background-color: rgba(255, 0, 0, 0);

background-color: rgba(255, 0, 0, .1);

background-color: rgba(255, 0, 0, .2);

background-color: rgba(255, 0, 0, .3);

background-color: rgba(255, 0, 0, .4);

background-color: rgba(255, 0, 0, .5);

background-color: rgba(255, 0, 0, .6);

background-color: rgba(255, 0, 0, .7);

background-color: rgba(255, 0, 0, .8);

background-color: rgba(255, 0, 0, .9);

background-color: rgba(255, 0, 0, .1);

En CSS3 también se introdujo una nueva forma de representar los colores con la denominación HSL (Hue, Saturation, Luminosity), es decir, Tono, Saturación y Luminosidad.

El Tono se indica con un valor que va del 0 a 360 que representa la totalidad de los colores existentes en la rueda de colores, la Saturación que va de 0% a 100%, en el que 0 es nada saturado y 100 % la saturación máxima, y la luminosidad que desde 0% (nada luminoso, o sea, negro) a 100% (máxima luminosidad, o sea, blanco), y al 50% representa al color puro.

También se le puede agregar el canal ALPHA. Te invito a que experimentes, interactuando con dicho sistema de colores en la página hslpicker.com en la que además, encontrarás el código de color EXADECIMAL, RGB y HSL.
